README - Opstarten van de NS Fietsenstalling

GIT repository: https://github.com/MarcBax/MiniProject_NS_fietsenstalling

1. Pak het ZIP bestand uit op je computer.
2. Start PyCharm op en importeer het project
3. Start het bestand Ns_fietsenstallen.py
4. (optioneel) Download de telegram app, en maak een account aan, om een bericht te krijgen als u uw fiets ophaalt.
